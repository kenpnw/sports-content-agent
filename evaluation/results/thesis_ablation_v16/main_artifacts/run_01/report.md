# 勇士客场三分雨险胜湖人

## Executive Summary

勇士凭借库里和汤普森的外线火力，在首节和次节建立领先，湖人虽在末节追平，但库里终场前8秒命中关键三分，勇士113-110险胜。

本场比赛勇士客场挑战湖人，最终以113-110险胜，比赛过程跌宕起伏。勇士凭借首节和次节精准的三分投射建立领先优势，而湖人在末节展开猛烈反扑，一度追平比分，但库里在终场前8秒命中关键三分，帮助勇士锁定胜局。以下从战术角度对关键回合进行详细分析。

首节中段，湖人通过战术配合得分。10分58秒，安东尼·戴维斯在弱侧利用防守者注意力被持球人吸引的瞬间，突然切入篮下接球完成扣篮。本回合属于『反跑』(Back-cut) 战术，戴维斯敏锐地捕捉到防守漏洞，轻松得分。然而，勇士的进攻随后出现失误：10分31秒，德雷蒙德·格林在弧顶持球时遭遇防守压迫，试图传给弱侧队友但传球路线被提前预判拦截，导致球权丢失。这是一次典型的传球失误，格林在高压下未能做出正确决策。
