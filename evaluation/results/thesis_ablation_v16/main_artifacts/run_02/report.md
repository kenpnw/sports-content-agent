# 勇士客场三分险胜湖人

## Executive Summary

勇士凭借库里末节8秒准绝杀三分以113-110险胜湖人。湖人开局由詹姆斯和戴维斯内外开花，但勇士汤普森和库里连续命中三分反超。湖人失误较多，关键时刻格林失误葬送机会。
